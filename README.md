# Object Detection

Detect objects in your video.
----------INSTRUCTIONS----------
1) Download main.py
2) Move the file to any folder
3) Get your video in mp4 format
4) Move video to the same folder as the script
5) Rename your video to video.mp4
6) Open cmd and run the next command:
pip install opencv-python ultralytics

7) After install, run:
cd path/to/the/script
Example: cd D:\object_detection
8) Then run:
python main.py

9) The video will start processing, it will take time. It all depends on what GPU you have.
10) The result will appear in a video called "result.mp4"
That's it!
